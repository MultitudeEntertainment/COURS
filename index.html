Bases de Python

1. Qu’est‑ce qu’un programme Python ?

Un programme Python est un fichier texte contenant des instructions. Python lit ces instructions de haut en bas et les exécute.

2. Premier programme

print("Bonjour, Python !")
print : fonction d’affichage ( ) : parenthèses " " : chaîne de caractères

3. Variables

nom = "Durand"
age = 17
moyenne = 13.5
admis = True
= affecte une valeur str : texte int : entier float : nombre à virgule bool : vrai/faux

4. Saisie clavier

prenom = input("Votre prénom : ")
print("Bonjour", prenom)

5. Conversion

texte = input("Entrez un nombre : ")
n = int(texte)
print(n + 10)

6. Types de base

  • int : entier
  • float : nombre décimal
  • str : texte
  • bool : True / False

7. Opérateurs

//  division entière
%   modulo
**  puissance

8. Expressions

print(c)

9. Commentaires

Conditions

1. Structure

if condition:
    ...
elif autre_condition:
    ...
else:
    ...

2. Comparaisons

==  égal
!=  différent
>   strictement supérieur
<   strictement inférieur
>=  supérieur ou égal
<=  inférieur ou égal

3. Positif / négatif

n = int(input("Entrez un entier : "))

if n > 0:
    print("positif")
elif n < 0:
    print("négatif")
else:
    print("nul")

4. Pair / impair

if n % 2 == 0:
    print("pair")
else:
    print("impair")

5. Intervalle

if 10 <= n <= 20:
    print("dans l'intervalle")

6. Conditions combinées

if age >= 18 and age <= 25:
    print("Jeune adulte")

7. Exercices (conditions)

  • Tester si un nombre est dans [5 ; 15]
  • Tester si un mot fait plus de 5 lettres

Boucles

1. Boucle for

for i in range(1, 6):
    print(i)

2. Boucle while

i = 0
while i < 5:
    print(i)
    i += 1

3. Somme de 1 à n

n = int(input("n : "))
s = 0
for i in range(1, n+1):
    s += i
print(s)

4. Boucles imbriquées

for i in range(3):
    for j in range(2):
        print(i, j)

5. Exercices (boucles)

  • Afficher les nombres de 1 à 100
  • Afficher les multiples de 7
  • Compter les lettres d’un mot

Listes & tableaux

1. Créer une liste

notes = [12, 15, 9, 18, 10]

2. Parcourir une liste

for n in notes:
    print(n)

3. Moyenne

m = sum(notes) / len(notes)

4. Compter les >= 10

c = 0
for n in notes:
    if n >= 10:
        c += 1

5. Grille 2D

grille = [
    [0,1,0],
    [1,0,1],
    [0,1,0]
]

6. Compter les 1

total = 0
for ligne in grille:
    for case in ligne:
        if case == 1:
            total += 1
