Bases de Python Chapitre 1

Objectifs : afficher du texte, utiliser des variables, lire au clavier, convertir des valeurs.

1. Afficher du texte avec print()

print("Bonjour !")
print("Bienvenue dans l'Académie Python")
print() est une fonction intégrée de Python qui affiche ce qu’on lui donne entre parenthèses. Le texte doit être entouré de guillemets " " ou ' '.

Erreurs fréquentes

  • Oublier une parenthèse : print("Salut" → SyntaxError
  • Oublier les guillemets : print(Salut) → NameError

2. Variables : stocker des informations

nom = "Samir"
age = 17
moyenne = 13.5
admis = True
Une variable est un nom qui pointe vers une valeur. On utilise le signe = pour faire une affectation (mettre une valeur dans une variable).
NomValeurType
nom"Samir"str (texte)
age17int (entier)
moyenne13.5float (décimal)
admisTruebool (vrai/faux)

Règles pour les noms de variables

  • Pas d’espace : utiliser _ → note_math
  • Pas de caractères spéciaux (%, -, ?, …)
  • Ne pas commencer par un chiffre
  • Python distingue majuscules/minuscules : age ≠ Age

3. Saisie clavier avec input()

prenom = input("Votre prénom : ")
print("Bonjour", prenom)
input() affiche un message, attend que l’utilisateur tape quelque chose, puis renvoie ce texte. Le résultat est toujours une chaîne de caractères (type str), même si l’utilisateur tape un nombre.

4. Conversion en entier avec int()

texte = input("Entrez un nombre : ")
n = int(texte)
print("n + 10 =", n + 10)
Si l’utilisateur tape autre chose qu’un nombre entier (par exemple "abc"), int() provoquera une erreur ValueError. On verra plus tard comment gérer ça proprement.

5. Résumé des notions

  • print() : afficher des choses
  • variable = valeur : stocker une information
  • input() : lire au clavier (toujours du texte)
  • int() : convertir du texte en entier

Boucles Chapitre 4

Objectifs : for, while, range, boucles imbriquées, cas pratiques.

1. Boucle for avec range()

for i in range(1, 6):
    print("i vaut", i)
range(1, 6) génère les nombres 1, 2, 3, 4, 5 (le 6 est exclu).

2. Boucle while

compteur = 0
while compteur < 5:
    print("Compteur =", compteur)
    compteur = compteur + 1
Attention aux boucles infinies : si la condition reste toujours vraie, la boucle ne s’arrête jamais.

3. Somme de 1 à n

n = int(input("Entrez un entier positif : "))

for i in range(1, n + 1):

print("Somme de 1 à", n, "=", somme)

4. Exercice

Exercice – Moyenne depuis un fichier
IntermédiaireFichiers + boucles.

Un fichier notes.txt contient une note par ligne. Lis le fichier, convertis chaque ligne en nombre, calcule la moyenne et affiche-la. Ignore les lignes vides.

5. Correction

somme = 0
compte = 0

with open("notes.txt", "r", encoding="utf-8") as f:
    for ligne in f:
        ligne = ligne.strip()
        if ligne != "":
            try:
                note = float(ligne)
                somme += note
                compte += 1
            except ValueError:
                print("Ligne ignorée :", ligne)

if compte > 0:
    print("Moyenne =", somme / compte)
else:
    print("Aucune note valide.")

Introduction aux objets Chapitre 9

Objectifs : comprendre l’idée de classe, objet, attribut, méthode.

1. Classe simple

class Joueur:
    def __init__(self, nom, vie):
        self.nom = nom
        self.vie = vie

    def afficher(self):
        print(self.nom, "a", self.vie, "points de vie.")

j1 = Joueur("Léa", 100)
j2 = Joueur("Hugo", 80)

j1.afficher()
j2.afficher()
Une classe est un modèle. Un objet est une "instance" de cette classe. self représente l’objet lui-même.

2. Exercice

Exercice – Classe Rectangle
IntermédiaireObjets + méthodes.

Crée une classe Rectangle avec largeur et hauteur, une méthode pour calculer l’aire, et une méthode pour l’afficher.

3. Correction

class Rectangle:
    def __init__(self, largeur, hauteur):
        self.largeur = largeur
        self.hauteur = hauteur

    def aire(self):
        return self.largeur * self.hauteur

    def afficher(self):
        print("Rectangle", self.largeur, "x", self.hauteur,
              "- aire =", self.aire())

r = Rectangle(3, 5)
r.afficher()

Projet 4 – Gestion simple de notes

notes = []

while True:
    texte = input("Entrez une note (ou 'fin') : ")
    if texte == "fin":
        break
    try:
        note = float(texte)
        notes.append(note)
    except ValueError:
        print("Note invalide, réessaie.")

if len(notes) == 0:
    print("Aucune note.")
else:
    somme = 0
    for n in notes:
        somme += n
    moyenne = somme / len(notes)
    print("Notes :", notes)
    print("Moyenne :", moyenne)
